### Архитектурные и тактические решения:

-Проект состоит из двух основных разделов и нескольких вспомогательных. Основными являются: scr (вспомогательные скрипты обучения модели и генерации предсказаний, пайплайны),
tests(тесты модулей и  процесса обучения). Вспомогательные директории посвящены обработке сырых данных, Eda и конфигам. 
- Для конфигурирования использовалась `hydra`.
- Созданы конфигурации для обучения логистической регрессии и k соседей.
- Для тестирования модуля обучения генерируются синтетические данные: для этого каждый столбец-признак исходных данных параметризуется. Далее на основании полученных параметров генерируются ситетические данные на основе свойств исходных.
- В проекте проведен "ручной" разведовательный анализ данных.
В дополнение к нему скприптом автоматически генеруется отчет в html форме
- Для некотрорых сущностей из конфигов были написаны датаклассы.
